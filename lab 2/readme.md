#  Personal Portfolio Website — Lab 2 (Web Dev I)

This is my *Lab 2 Web Development I assignment, where I built a **personal portfolio website* using *HTML* and *external CSS*.  
The project focuses on applying core CSS concepts such as:

- External Stylesheet (style.css)
- Color themes
- Typography using Google Fonts
- Navigation bar styling
- Box model (margin, padding, border)
- Buttons & hover effects
- Tables with proper styling
- CSS positioning (sticky header, fixed button)
- Clean layout separation using <hr>

---

Project Overview*

The portfolio contains the following sections:

- *Header / Navigation Bar*
- *Hero Banner*
- *About Section*
- *Skills Table*
- *Projects Section*
- *Contact Form*
- *Back-to-Top Button*
- *Footer*

The design uses a *consistent color theme* and *Poppins Google Font* for a modern and professional appearance.

 Learning Outcomes*

✔ Apply an external CSS stylesheet  
✔ Use class, ID, and element selectors
